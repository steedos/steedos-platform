## Getting Started

- 注册 Datadog 账户。
- 创建 .env.local 文件，并配置环境变量。

```
DATADOG_API_KEY=xxx
```

- 运行开发服务器:

```bash
docker-compose up
yarn
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
